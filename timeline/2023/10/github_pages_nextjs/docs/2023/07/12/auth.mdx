## OAuth

### OAuth 2.1
- OAuth 2.1 是对 OAuth 2.0 的改进，简化了规范，移除了不安全的功能（如隐式授权）
- 更加注重安全性，推荐使用授权码模式（Authorization Code Flow）并结合 PKCE（Proof Key for Code Exchange）

### OpenID Connect (OIDC)

- 基于 OAuth 2.0 的身份层协议，提供用户身份验证功能。
- 广泛用于单点登录（SSO）和用户信息共享。
